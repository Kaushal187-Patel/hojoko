# HOZOKO

HOZOKO is an online store with a public shopping site and separate admin areas for the store owner and sellers. Shoppers can browse, buy, and manage orders; admins manage the full catalog; sellers manage only their own products.

## Features

### Storefront (customers)
- Signup, login, forgot / reset password
- Product browse, search, categories, and product details
- Cart, wishlist, and checkout
- Orders, profile, and saved addresses
- Limited edition products, recently viewed, cookie consent
- Support / help center

### Main admin
- Store dashboard and overview
- Manage all products from every seller
- Categories, homepage content, orders, and customers
- Create seller accounts
- Activate or deactivate seller accounts

### Seller admin
- Separate seller dashboard
- Add, edit, and remove **only their own** products
- View their product count and low-stock alerts
- Cannot see or change other sellers’ products

## User roles

| Role | What they can do |
|------|------------------|
| Customer | Shop, cart, checkout, profile |
| Main admin | Full admin panel |
| Seller | Seller panel — own products only |

If you are upgrading an older install, run the seller migration script once (see Getting Started).
